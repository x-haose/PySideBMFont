
# PySideBMFont

基于python side6的图片字体制作工具，完成了基本的功能，使用`PySide6-Fluent-Widgets`库做的界面，还挺美观的


## 在本地运行

Clone 这个 project

```bash
  git clone https://github.com/x-haose/PySideBMFont.git
```

前往项目目录

```bash
  cd PySideBMFont
```

安装依赖

```bash
  pdm sync
```

本地运行

```bash
   pdm run python -m bmfont
```

代码检查

```bash
   pdm run check
```

构建

```bash
   pdm run build
```


## 技术栈

- Pillow
- PySide6-Fluent-Widgets
